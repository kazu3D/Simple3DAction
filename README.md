### 【Simple3DAction】

## 🎮 ゲーム概要

Unityで制作中のシンプルな3Dアクションゲームです。
3Dゲーム制作の基礎理解を目的としており、プレイヤー操作や物理挙動を中心に実装しています。
※ 本作品は現在も制作途中の学習用プロジェクトです。


## 🛠 実装内容（対応可能な作業）

- プレイヤーの移動・ジャンプ操作（Input System）
- CharacterController を使用したプレイヤー移動・衝突判定
- カメラ追従処理
- 地面判定・落下処理
- シンプルな3Dステージ構成


## 📚 使用技術・環境

- Unity 6
- C#
- Input System
- CharacterController
※本プロジェクトでは、安定したキャラクター操作を目的として
　CharacterControllerを使用しています。


## ✍️ 工夫した点・学習内容

- CharacterController を用いたジャンプ・落下制御の実装
- 入力処理と移動処理を分離し、可読性を意識した実装
- CharacterController による3D空間での当たり判定・接地判定の理解
